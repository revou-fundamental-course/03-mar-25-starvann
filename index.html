About Me

Saya Azaria Adila Putri, siswa SMK jurusan Pengembangan Perangkat Lunak dan Gim (PPLG), dengan minat besar dalam teknologi dan pengembangan website serta game. Saya sedang mempelajari berbagai teknologi pemrograman seperti HTML, CSS, JavaScript, GDScript, dan Python, serta mendalami desain UI/UX agar hasilnya menarik dan fungsional.

Saat ini, saya sedang mengerjakan beberapa proyek, termasuk pembuatan website perusahaan, pengembangan game RPG 2D dengan pixel art, serta eksplorasi dalam pengembangan game 3D menggunakan Unity.

Ke depan, saya berharap dapat berkarir sebagai Game Developer atau Softwere Engineer yang menciptakan suatu produk yang bermanfaat di bidang game maupun perangkat lunak. Saya juga bercita-cita membangun studio game independen di masa depan, dengan fokus pada kreativitas dan teknologi terbaru untuk menghadirkan pengalaman yang menarik dan bermakna bagi pengguna.
